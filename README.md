# Multi-Tier AI Fraud Detection System

## Overview
TIERA is a web-based AI-powered fraud detection dashboard designed to monitor supplier transactions and detect suspicious activities using a multi-factor risk scoring engine.

The system analyzes transaction patterns, assigns risk scores, and visualizes fraud insights through an interactive dashboard.

---

## Features

-  Secure Login System
-  Real-Time Fraud Dashboard
-  Multi-Factor Risk Scoring Engine
-  Interactive Charts (Chart.js)
-  Fraud Alert Tracking
-  Export Fraud Reports to PDF
-  SQLite Database Integration

---

## Tech Stack

- Backend: Python (Flask)
- Database: SQLite
- Frontend: HTML, CSS, JavaScript
- Visualization: Chart.js
- Reporting: ReportLab (PDF Export)
